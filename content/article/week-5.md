### Java 是 FP?

Java 8 之后，Java 新加入的 lambda 表达式，Optional 跟 Stream，还有双冒号，差不多直接把 Java 变成函数式语言了 (大雾).

简单体验一下：

```java
    /**
     * 递归查找符号
     * @param name 符号的名字
     * @return 查找到的符号，若符号不存在则为 null 
     */
    public Optional<Variable> get(String name) {
        assert name != null;

        return getInCurrent(name)
            .or(() -> parent.flatMap(scope -> scope.get(name)));
    }
```

```java
    public Optional<Variable> lookupVariable(String name) {
        return lookupWithClass(Variable.class, name);
    }

    public Optional<Function> lookupFunction(String name) {
        return lookupWithClass(Function.class, name);
    }

    private<T extends Symbol> 
    Optional<T> lookupWithClass(Class<T> cls, String name) {
        return lookup(name)
            .filter(cls::isInstance)
            .map(cls::cast);
    }
```
